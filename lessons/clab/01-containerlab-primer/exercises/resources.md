## Link to interesting community lab repo with brief description

- [GitHub - srl-labs/srl-telemetry-lab: Interactive Streaming Telemetry lab with Nokia SR Linux nodes forming a Clos topology · GitHub](https://github.com/srl-labs/srl-telemetry-lab)

This repo spins up a Clos fabric as well as a streaming telemetry stack with gnmic, prometheus and grafana. It is a demonstration of how observability and logging tools can be used within a network.
